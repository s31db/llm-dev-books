# LLM-Assisted Software Design, a Pattern Language of New Development Practices

Ce dépôt contient le manuscrit, les illustrations et les annexes du livre *LLM-Assisted Software Design, a Pattern Language of New Development Practices*.

## Description

Cet ouvrage explore les nouvelles pratiques de conception logicielle à l’ère des modèles de langage (LLM). Il propose un langage de motifs pratiques, illustrés, à destination des développeurs, architectes, coachs, formateurs et chercheurs curieux de l’artisanat logiciel augmenté.
